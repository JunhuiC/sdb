The Sequoia scripts found in this directory are init.d script that 
can be used with a SuSE installation.

- sequoia.suse: this is the script you should use. Do not forget to update the
 variables at the beginning of the script to provide your installation 
 directory as well as virtual database information.
 
- sequoia.old: is an older contribution for SuSE that was compatible with the
 early releases of Sequoia 2.x.
